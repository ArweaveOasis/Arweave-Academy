## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/zoidkiller/Arweave-Academy.git
   ```
2. Navigate to the project directory:
   ```bash
   cd XHunterAirdrop
   ```
3. Open `index.html` in your web browser.
